# Felipe

Software engineer. Co-author of [Hall of Impossible Dreams](https://hallofdreams.org), a blog about code, fiction, nonfiction, and poetry.

## What I do

I build things at an AI company in the finance space. Before that, I was at GitHub. I've been writing software for over a decade, mostly in the space where ML, optimization, and systems programming overlap.

Outside of work, I co-maintain hallofdreams.org with [Dave](https://hallofdreams.org/tabs/about/), where we write long-form technical deep dives, fiction, and the occasional poem about competitive programming.

## Notable projects

**[HATETRIS World Record](https://github.com/thecog19/HATETRIS-public-beamsearch)** — Current world record holder (with Dave) in [HATETRIS](https://qntm.org/hatetris), the hardest version of Tetris ever made, at thousands of points. Built a Rust emulator averaging ~5μs/move, a beam search engine, and an NNUE neural network. The story is told across two posts: [Getting the World Record](https://hallofdreams.org/posts/hatetris/) and [Losing the World Record](https://hallofdreams.org/posts/hatetris-2/). Featured on the [CoRecursive podcast](https://corecursive.com/hatetris-with-david-and-felipe/).

**[The x86 Inferno](https://hallofdreams.org/posts/x86-inferno/)** — Solved all of Advent of Code 2025 in x86-64 assembly, then wrote a blog post about the experience structured as a descent through Dante's *Inferno*.

**[Agents and Ants](https://hallofdreams.org/posts/agents-and-ants/)** — A post-mortem of the SWARM distributed ant colony simulation challenge, covering multi-agent AI coding workflows and the technical evolution of an ant brain.

**[Trackmania ML Series](https://hallofdreams.org/posts/trackmania-1/)** — A three-part series on the history and practice of machine learning applied to Trackmania, from supervised learning on screenshots to modern reinforcement learning.

**[SAArchiver](https://github.com/thecog19/SAArchiver)** — A Rails API for archiving Something Awful threads, because the only way to make sure content survives on the internet is to host it and back it up.
